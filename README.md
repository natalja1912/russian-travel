# Название проекта
**Путешествия по России**
Проект рассказывает о интересных местах в России и о том, как можно туда попасть

## О чем проект
 * Чего мы не видели в России?

 * Кольский полуостров
 * Алтай
 * Зимний Байкал
 * Карелия
 * Как добраться до Байкала
